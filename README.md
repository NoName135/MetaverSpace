## 功能

### 前台
- [x] 瀏覽文章
- [x] 瀏覽、篩選商品
- [x] 商品加入購物車
- [x] 商品加入收藏清單
- [x] 購物車及收藏清單商品移動
- [x] 商品結帳、輸入優惠碼
- [x] 送出訂單、付款
- [x] 交易記錄查詢
- [x] 預約平台表單
- [x] 預約查詢、取消預約
- [x] 留言板

### 後台
- [x] 登入 / 登出
- [x] 商品編輯、刪除
- [x] 商品選擇配件
- [x] 訂單編輯、刪除
- [x] 訂單圖表統計
- [x] 文章編輯、刪除
- [x] 預約編輯、刪除
- [x] 優惠券編輯、刪除
- [x] 留言板檢視、刪除
